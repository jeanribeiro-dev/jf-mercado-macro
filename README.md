# JF Mercado Macro Dashboard

Coloque aqui a sua planilha (em formato CSV, Excel ou similar) para que possamos incorporá-la e visualizá-la no dashboard de forma harmônica.

## Estrutura sugerida:
- Cole o arquivo da planilha nesta pasta.
- Nós processaremos os dados para separar cada estratégia e fazer a consolidação (resultados, drawdown/rebaixamento).
